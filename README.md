# AEROCLEAN
## The Website to check up on the air.
Our website allows users to be able to see statistics about the air and what people can do to reduce air pollution.
We also provide services to help with those actions, such as an eco-frienly Commuter Routing, CO2 Calculator, and even a real-time AQI Checker to check the air quality in cities such as Jakarta, Surabaya, Semarang, Yogyakarta, and Bandung.
